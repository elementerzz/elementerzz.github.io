Noob's Difficulty Chart Obby

Creator at The Noob's

Noob's Difficulty Chart Obby Project!

This game is a classic Difficulty Chart Obby. Players make their way through challenging stages that progressively get harder. There are multiple difficulties and there are still some more to come!

For this game I am the the Main Scripter and Main Builder. Others have helped with some of the stages as well. I scripted the main backbone of the game and everything else such as GUI, Stage Transfer etc. I also made most of the stages. I also made all of the unique obby parts. An interesting part about them is that, I made a script that makes it so, if you name a part a specific name, it becomes that unique obby part.

e.g If you name a part "SpinningPart", and inside the part you put an axis value determining which axis it is to spin on and a speed value determining the speed, it automatically makes the part work without the need of adding any more scripts.

It can be done with many other obby parts as well such as: "PusherPart", "KillPart", "DamageBrick" etc.

I made a game that showcases all of this and you can see it here.

This game is still an ongoing project so more stages are to come soon!
